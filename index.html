What it does

Every button does something real.

✂️
CUT & Fade

Instant CUT or configurable Mix crossfade. Set the fade duration. Hit the button. Done.

🎥
Camera control

Send any camera to Preview with one press. Buttons turn green on preview, red when live. Set your ATEM IP once — every button shares it automatically.

⏺
REC & STREAM

Dedicated record and stream toggle buttons. Red when active, dark when off.

⏱
Auto Director

Automatic cuts on a randomized timer. Choose which cameras to include, set per-camera dwell times. Always picks a fresh angle — never re-cuts the same camera.

🎤
Audio Director

Detects who's speaking via your audio interface and cuts to the right camera automatically. Auto-installs Python dependencies on first run — no setup needed.

🖼
Media Player

Load stills into MP1 and send to preview in one press. Shows which still is loaded on the button.

FAQ

Quick answers.

Which ATEM models are supported?

Tested on ATEM Mini Pro ISO. Expected to work on ATEM Mini and Mini Pro (same protocol). ATEM Mini Extreme and G2 support is planned.

What Stream Deck do I need?

Any Stream Deck model works — MK1, MK2, XL, Mini, or +. macOS only for now. Windows support is coming.

How does Audio Director work?

You need a multi-channel audio interface (like a Zoom L8) connected to your Mac. ATEMdirector listens to individual channels and cuts to the matching camera when it detects voice activity.

Is this affiliated with Blackmagic Design?

No. ATEMdirector is an independent product not affiliated with or endorsed by Blackmagic Design.

Do I need coding knowledge to set it up?

No. Installation takes about 5 minutes. Audio Director auto-installs all required Python packages on first run. A step-by-step guide is included and email support is available if you get stuck.

How much does it cost?
